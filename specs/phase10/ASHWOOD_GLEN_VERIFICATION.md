# Phase 10 Ashwood Glen Manual Verification

Use the Phase 3 Elder Road scene.

## Transition Path

- [ ] Start a new run.
- [ ] Travel to the Elder Stone before finishing The Elder Road.
- [ ] Verify Enter Ashwood Glen is unavailable or blocked.
- [ ] Complete The Elder Road objectives through the bandit and Elder Stone.
- [ ] Return to the Elder Stone.
- [ ] Verify Enter Ashwood Glen is available.
- [ ] Enter Ashwood Glen.
- [ ] Verify the header shows Ashwood Glen.
- [ ] Verify the map changes to a 5x5 Ashwood layout.
- [ ] Use Return Road.
- [ ] Verify Elder Road Outskirts is restored.
- [ ] Re-enter Ashwood Glen and verify visited/cleared state persists.

## Hazards

- [ ] Move onto Burning Thorn.
- [ ] Verify it triggers once and deals nonlethal damage.
- [ ] Raise defense to 4 or higher, then verify Burning Thorn damage is mitigated in a fresh run.
- [ ] With Coal Remembers Flame discovered and active, verify the resonance mitigation message appears.
- [ ] Interact with Cinder Pool.
- [ ] Verify it drains mana for Ember Sage when mana is available.
- [ ] Verify it drains health for non-Ember Sage classes or when mana behavior does not apply.
- [ ] Verify Cinder Pool cannot reduce health below 1.

- [ ] Defeat Cinder Acolyte.
- [ ] Verify XP and loot are granted once.
- [ ] Open Cinder Cache.
