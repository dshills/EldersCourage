# Phase 9 Item Resonance Manual Verification

Use the Phase 3 Elder Road scene. Start fresh for each class pass when practical.

## Ember Sage Primary Path

- [ ] Start a new run as Ember Sage.
- [ ] Acquire Ashen Ring from the Weathered Shrine.
- [ ] Equip Ember Staff and Ashen Ring.
- [ ] Open inventory and inspect either item.
- [ ] Verify a hinted Coal Remembers Flame resonance appears without exposing hidden cursed text.
- [ ] Use Ember Bolt while both items are equipped.
- [ ] Verify Coal Remembers Flame is discovered.
- [ ] Verify Ember Bolt damage increases from the discovered resonance.
- [ ] Unequip one required item.
- [ ] Verify the discovered resonance appears inactive and its bonus no longer applies.

## Roadwarden Path

- [ ] Start a new run as Roadwarden.
- [ ] Equip Roadwarden's Notched Blade and Roadwarden Vest.
- [ ] Identify or partially reveal the Notched Blade as needed.
- [ ] Use Guarded Strike.
- [ ] Verify Road Oath Tempered can reveal.
- [ ] Verify the resonance gives the expected defensive benefit while both items are equipped.
- [ ] Unequip either required item and verify the effect stops.

## Gravebound Scout Path

- [ ] Start a new run as Gravebound Scout.
- [ ] Equip Scout Knife and Whisperthread Cloak.
- [ ] Defeat an enemy with the required gear equipped.
- [ ] Verify Grave Thread reveals.
- [ ] Verify Piercing Shot cost is reduced while the resonance is active.
- [ ] Confirm the item details show the discovered resonance without exposing unrelated hidden combinations.

## Breath Debt Cursed Path

- [ ] Acquire Ashen Ring and Elder Glass Charm in the same run.
- [ ] Equip Ashen Ring and carry Elder Glass Charm.
- [ ] Trigger Ashen Ring Blood Price through skill use.
- [ ] Verify Breath Debt reveals as cursed.
- [ ] Use a skill while Breath Debt is active.
- [ ] Verify the added health cost is applied.
- [ ] Verify the health cost cannot reduce the player below 1 health.

## Merge Path

- [ ] As Ember Sage, discover Coal Remembers Flame.
- [ ] Reveal Varn's name on Ashen Ring.
- [ ] Raise Ashen Ring attunement to the merge requirement.
- [ ] Open inventory and inspect Ashen Ring or Ember Staff.
- [ ] Verify the merge hint is visible and marked ready.
- [ ] Press Merge.
- [ ] Verify the Item Merge panel lists source items, result, and requirements.
- [ ] Press Prepare.
- [ ] Verify the confirmation state is required before completion.
- [ ] Press Confirm Merge.
- [ ] Verify Ashen Ring and Ember Staff are consumed.
- [ ] Verify Staff of the Ashen Orator is added and equipped.
- [ ] Verify Varn's revealed soul state is transferred to the staff.
- [ ] If Ashen Ring Blood Price had been revealed, verify the carried risk appears on the staff.

## UI Checks

- [ ] Resonance text in item details remains readable at the current inventory panel size.
- [ ] Hidden and cursed resonances do not reveal from inventory browsing alone.
- [ ] Merge panel can be cancelled.
- [ ] Escape closes a pending merge instead of leaving stale state.
- [ ] Existing bargain panel behavior is unchanged.
- [ ] Message log shows resonance and merge messages with distinct labels and colors.
